# Test Mini App

This repository contains the **Test Mini App**, a lightweight application designed for testing and experimentation purposes.

## Features

- Simple and modular codebase.
- Easy to set up and run.
- Designed for quick prototyping and testing.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/test-mini-app.git
    ```
2. Navigate to the project directory:
    ```bash
    cd test-mini-app
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

Start the application:
```bash
npm start
```
